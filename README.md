# Loan Risk & Portfolio Analysis Project

## Overview
This project analyzes a multi-asset loan portfolio including residential, auto, student, and commercial loans. The goal is to evaluate risk and performance using key financial metrics.

## Tools Used
- SQL (MySQL)
- Power BI
- Excel (data preparation)

## Key Metrics
- Debt-to-Income Ratio (DTI)
- Loan-to-Value Ratio (LTV)
- Credit Score
- Loan Status
- Investor Type

## Key Analysis
- Risk segmentation (High vs Low risk loans)
- Portfolio distribution by loan type
- Investor exposure analysis
- Payment performance tracking

## Business Value
Helps financial institutions identify risky loans, manage portfolio exposure, and support investment decisions in structured finance environments.
